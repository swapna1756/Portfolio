# **App Name**: Aura

## Core Features:

- Bubble Title Scene: Interactive landing screen featuring large 3D inflated typography with glossy pink textures and soft neon glow effects.
- Dynamic Tagline Generator: An AI-powered tool that suggests elegant and poetic portfolio subtitles based on the user's specific creative industry.
- Flutter Animation Engine: A lightweight animation system that generates glowing butterflies following smooth, cinematic flight paths with sparkle particle trails.
- Glimmer Cursor: Custom butterfly-themed cursor interaction that emits a soft trail of glowing particles as the user moves across the interface.
- Cinematic Entrance Sequence: High-fidelity transition including zoom-blur and motion-blurred opacity fades triggered by a glassmorphism CTA.
- Atmospheric Aurora: A high-performance CSS and Canvas-driven background with moving light blobs and deep purple gradients.

## Style Guidelines:

- A dark, luxury scheme inspired by twilight aesthetics. Primary color is a vibrant Bubblegum Pink (#FF4DA6) for a glowing title effect. Background is a deeply desaturated Dark Plum (#161214) to enhance the neon contrast.
- The accent color is a lush Radiant Orchid (#BF40BF), 30 degrees left on the color wheel to provide analogous depth to gradients.
- A font pairing featuring 'Poppins' for headlines to maintain a geometric, avant-garde feel, and 'Inter' for small descriptive text to provide machined clarity and elegance.
- Minimal luxury composition with strict center-aligned typography and ample negative space to create a 'cinematic screen' feel.
- Iconography consists of delicate, glowing organic shapes and ethereal particles rather than literal UI symbols.
- Dreamy and smooth movements featuring GSAP-powered staggering and jelly-effect hover interactions for the primary CTA button.
